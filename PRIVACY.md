# Privacy Policy

LiftCard is a local-first Chrome extension.

## Data Collection
LiftCard does **not** collect, transmit, or store any personal data on external servers.

## Local Storage
All data (workouts, client packs, reports) is stored **locally** on the user’s device using:
- `chrome.storage.local`
- `localStorage` (for standalone client files)

No data ever leaves the user’s computer unless the user explicitly exports or shares a file.

## Network Requests
LiftCard does **not** make any network requests.
There are:
- no APIs
- no analytics
- no trackers
- no third-party services

## Accounts
LiftCard does **not** require accounts, logins, or authentication.

## File Sharing
Client Packs and Report Packs are exported as standalone HTML files.
Sharing (e.g. via WhatsApp, email, or other tools) is entirely controlled by the user.

## Contact
If you have questions about privacy, you can contact the author via GitHub.
